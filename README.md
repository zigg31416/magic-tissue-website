# Magic Tissue — ম্যাজিক টিস্যু

Single-page Bengali ecommerce landing page built with Vite + React + TypeScript.

## Local development

```bash
npm install
npm run dev
```

The dev server runs at http://localhost:5173.

## Production build

```bash
npm run build
npm run preview
```

## Vercel deployment

Vercel settings are included in `vercel.json`:

- Framework Preset: `Vite`
- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: `dist`

## Project structure

- `index.html` — entry HTML with Bengali Google Fonts (Hind Siliguri) and Inter
- `src/main.tsx` — React entry
- `src/App.tsx` — single-file landing page composed of sections (PromoBar, Header, Countdown, Hero, Pricing, Steps, Reviews, FAQ, FinalCTA, Footer)
- `src/styles.css` — full responsive stylesheet
- `public/assets/` — local product images (downloaded from the reference site)
